# Slackker

Slack client like Twitter

## Description

Slackker is Slack client like Twitter.
This client to stream posts.

## Requirement

- Node.js

## Usage

```
npm start
```

## installation

```
git clone https://github.com/tanishi/Slackker
cd Slackker
npm install
```
